# articles

This is a repository for my Zenn articles.

By pushing articles to this repository, the articles will be published on [Zenn](https://zenn.dev/gae) automatically.

## Usage

### Create a new article

```bash
$ npx zenn new:article
```

### Preview

```bash
$ npx zenn preview
```

### Publish

```bash
$ npx zenn publish
```
